BUG FIXES DETAIL
----------------

BUG 1: Interface Cutoff Too Restrictive
  Location: marangoni.cu:65, 188
  Change: 0.01 < f < 0.99  -->  0.001 < f < 0.999 (configurable)
  Impact: 10x more interface cells captured

BUG 2: Gradient Limiter Too Aggressive
  Location: marangoni.cu:31, 109-114
  Change: Hardcoded 5e8 K/m  -->  Configurable parameter
  Impact: Allows proper high-gradient LPBF simulations

BUG 3: Hardcoded Material Constants
  Location: marangoni.cu:196-210
  Change: T_MELT=1923K hardcoded  -->  Configurable T_melt, T_boil
  Impact: Multi-material support (Ti, SS, Al, etc.)

BUG 4: Unused Parameter
  Location: marangoni.cu:349-356
  Change: Removed unused length_scale from computeMarangoniVelocity()
  Impact: Cleaner API, correct velocity formula

BUG 5: Documentation Inconsistency
  Location: marangoni.h:16, 61
  Change: Clarified force formula notation
  Impact: Better code understanding
